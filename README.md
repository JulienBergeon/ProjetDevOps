### First time setup 

#### Copy .env file from example
```shell
cp .env.example .env
```

#### Installing Composer Dependencies For Existing Applications
```shell
composer update
```

#### Generate Key
```shell
php artisan key:generate
```

#### Node
```shell
npm i 
```

#### Launch
```shell
php artisan serve
```
